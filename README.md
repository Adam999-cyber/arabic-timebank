# نظام بنك الوقت العربي

مشروع بوت تليجرام لإدارة بنك الوقت باللغة العربية.

## المتطلبات

- Node.js
- حساب بوت في تليجرام
- مفتاح Google Gemini AI

## التشغيل

```bash
npm install
npm start
```

## الإعداد

1. احصل على توكن البوت من @BotFather
2. احصل على مفتاح Gemini من Google AI Studio
3. أضف المفاتيح في ملف bot.js
